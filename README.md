# Windows Server Backup- und Monitoring-Dokumentation

Dieses Repository enthält eine Schritt-für-Schritt-Dokumentation zur
Installation und Nutzung von Backup- und Monitoring-Lösungen auf einem
bereits installierten Windows Server.

Der Fokus liegt auf der praktischen Einrichtung, Grundkonfiguration
und dem grundlegenden Einsatz der eingesetzten Software.
Die Installation des Windows Servers selbst ist **nicht** Bestandteil
dieser Dokumentation.

---

## Projektumfang

Diese Dokumentation umfasst folgende Themen:

- Installation und Grundkonfiguration von **Hasleo Backup Suite**
- Installation und Grundkonfiguration von **Acronis True Image Backup**
- Praktischer Vergleich beider Backup-Lösungen
- Einführung in **PRTG Network Monitor**
- Installation und erste Inbetriebnahme von PRTG
- Grundlegende Nutzung von PRTG zur Systemüberwachung

---

### Hasleo Backup Suite
- Installationsablauf
- Erste Konfiguration
- Beispielhafte Backup-Jobs
- Grundlegende Wiederherstellungstests

### Acronis True Image Backup
- Installationsablauf
- Erste Konfiguration
- Beispielhafte Backup-Jobs
- Grundlegende Wiederherstellungstests

Der Fokus liegt auf der praktischen Anwendung, nicht auf einer

---

## Monitoring mit PRTG

Im Rahmen dieses Projekts wird **PRTG Network Monitor** zur Überwachung
des Systems eingesetzt.

Behandelte Themen:
- Installation und Grundkonfiguration
- Hinzufügen von Geräten
- Hinzufügen und Verwalten von Sensoren
- Überwachung grundlegender Systemparameter

Beispielhaft überwachte Parameter:
- CPU-Auslastung
- Freier Speicherplatz
- Netzwerkverbindung
- Internetzugang
- DNS-Erreichbarkeit

Die Dokumentation beschränkt sich bewusst auf grundlegende Monitoring-
Funktionen und einfache Sensoren.
